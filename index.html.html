Le constat terrain

Avant : papier,
retard, chaos.

Registres papier obsolètes Saisie manuelle, erreurs fréquentes, données non fiables. Reporting avec des semaines de retard.
Convois sans suivi Impossible de savoir où est le camion, si la livraison a été effectuée, ou d'alerter en temps réel.
Correspondances perdues Courriers et documents officiels sans traçabilité. Délais de transmission de 3 à 7 jours.
Zéro vision nationale Chaque magasin opère en silo. La Direction Générale attend les rapports mensuels papier.
La solution OPVN App

Après : digital,
précis, instantané.

01
Base de données locale (WatermelonDB) Fonctionne sans Internet. Toutes les données disponibles offline, synchronisées dès le retour en réseau.
02
Entrepôts en 3D avec React Three Fiber Visualisez chaque magasin avec vrais sacs (25/50/100 kg) empilés sur palettes selon le stock réel.
03
Cartographie nationale MapLibre Tous les magasins sur une carte. Un tap → détails → vue 3D. Navigation fluide entre régions.
04
Synchronisation cloud Supabase Statut visible : Synchronisé · En attente · Erreur. Les données circulent entre toutes les régions.
05
Design iOS premium glassmorphism Interface fluide, animations natives, dark mode total. La même qualité qu'une app Apple Store.

Prise en main

5 étapes pour
démarrer.

ÉTAPE 01
Installation
APK Android ou App Store iOS. Compatible Android 7+ et iOS 15+. Aucune configuration réseau requise.
👤
ÉTAPE 02
Création de compte
Nom complet, téléphone, email, PIN à 4 chiffres. 30 secondes pour être opérationnel.
🏗️
ÉTAPE 03
Configurer les magasins
Ajoutez vos entrepôts avec dimensions réelles. La scène 3D se génère automatiquement.
📦
ÉTAPE 04
Gérer les stocks
Ajoutez, retirez, transférez. Chaque mouvement est tracé, daté et attribué à un opérateur.
🌐
ÉTAPE 05
Synchroniser
Renseignez vos credentials Supabase. Toutes les régions partagent les données en temps réel.
